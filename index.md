# Air Controller

Use your Android phone as a controller for your PC.

## Downloads

- Android APK: [Air-Controller-Android-debug-a268060.apk](https://github.com/Arkane-o7/Air-Controller/releases/download/v0.1.0/Air-Controller-Android-debug-a268060.apk)
- Windows server: [Virtual-GamePad-Windows.zip](https://github.com/Arkane-o7/Air-Controller/releases/download/v0.1.0/Virtual-GamePad-Windows.zip)
- Linux server: [Virtual-GamePad-Ubuntu.zip](https://github.com/Arkane-o7/Air-Controller/releases/download/v0.1.0/Virtual-GamePad-Ubuntu.zip)
- Checksums: [SHA256SUMS.txt](https://github.com/Arkane-o7/Air-Controller/releases/download/v0.1.0/SHA256SUMS.txt)
- Full release notes: [Air Controller v0.1.0](https://github.com/Arkane-o7/Air-Controller/releases/tag/v0.1.0)

## Quick Start

1. Install the Android APK on your phone.
2. Extract and start the PC server:
   - Windows: run `VGamepadPC.exe`
   - Linux: `chmod +x bin/VGamepadPC && ./bin/VGamepadPC`
3. Keep phone and PC on the same network.
4. Connect from the app by scanning the server QR code or entering IP and port.
